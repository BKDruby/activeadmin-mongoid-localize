# Activeadmin::Mongoid::Localize



## Installation

Add this line to your application's Gemfile:

    gem 'activeadmin-mongoid-localize'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install activeadmin-mongoid-localize
